# 建築執照申請文件

## 使用情境
- 建築師準備建築執照申請文件時
- 明確各類文件格式與內容要求
- 確保申請文件符合主管機關標準

## 學習目標
- 掌握建築執照申請所需文件類型
- 理解文件格式與簽章要求
- 熟悉標準文件數量與提交流程

## 實務應用
### 標準申請文件列表：
1. **建築物申請書**
   - 建築物名稱、地址
   - 營業項目與建築用途
   - 申請人資訊

2. **地籍圖謄本**
   - 建築物所在位置
   - 地號、地目、地權屬性

3. **建築物設計圖說**
   - 建築物平面圖（1:50 or 1:100）
   - 結構圖、立面圖、剖面圖
   - 各樓層功能配置圖

4. **結構計算書**
   - 土壤承載力報告
   - 結構設計計算書
   - 抗風、抗震計算

5. **消防設備設置計畫**
   - 火災自動警報系統圖
   - 消防水系統圖
   - 疏散路線圖

6. **障礙設備設置圖**
   - 障礙設備配置圖
   - 出入口寬度與坡度
   - 設備使用空間

7. **環境影響評估報告（如需要）**
   - 噪音影響評估
   - 空氣品質影響評估

### 文件格式與要求：
- **圖面格式**：A0 或 A1，比例 1:50 或 1:100
- **文件種類**：PDF 格式，需可編輯
- **簽章要求**：需有建築師、結構工程師簽章
- **提交數量**：3 份（1 份正本，2 份副本）
- **提交方式**：線上系統或現場遞交
